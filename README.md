# Knight-Game

Functions:
* Player controller(left-right movement)
* Jump(hold to jump higher)
* Gravity
* fall gravity multiplier
* Camera(following player)
* slip n slide
* Main menu
* Death Screen



Animation:
* idle
* moving
* jump
